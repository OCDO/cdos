## Ontologies

CDOS contains the ontologies that describe crystallographic defects, following a modular approach according to the common classification of defects.

![Schematic representation of CDCO](docs/source/_static/CDCO.png)

 * [CDCO - Crystallographic Defect Core Ontology](https://ocdo.github.io/cdco/)
 * [PODO - Point Defect Ontology](https://ocdo.github.io/podo/)
 * [LDO - Line Defect Ontology](https://ocdo.github.io/ldo/)
 * [PLDO - Plane Defect Ontology](https://ocdo.github.io/pldo/)
